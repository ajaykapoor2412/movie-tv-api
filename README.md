Prerequisite
============================
1. Java || Version 1.8
2. Scala || Version 2.11.12
3. SBT || Version 1.3.8

Steps to run
============================
1. git clone https://github.com/ajaykapoor2412/movie-tv-api.git
2. cd movie-tv-api
3. sbt compile
4. sbt run
